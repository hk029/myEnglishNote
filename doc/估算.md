- 百分比分数化

  14-15% = 1/7

  16-17% = 1/6

  19-21% = 1/5

  24-26% = 1/4

  32-34% = 1/3

- **比重增加问题**：A同比增长a-1，B同比增长b-1，今年的A/B相比去年A/B增长多少

  A/B - (A/a)/(B/b) = A/B (1-b/a) = **A/B * (a-b)/a**

  我们还可以总结出一个规律：答案一定是小于A/B 和 a-b 的。（当a >1时，一般是这种情况。）
